# Get started with Microsoft.Maui.Essentials.AI

`Microsoft.Maui.Essentials.AI` provides two main capabilities, each with its own installation and registration steps. Choose the guide that matches what you want to build:

| Guide | What you build | Minimum OS |
|-------|---------------|------------|
| [Chat client](getting-started-chat.md) | Conversational AI with streaming, tool calling, and structured output using Apple Intelligence | iOS 26.0+ / macOS 26.0+ |
| [Text embeddings](getting-started-embeddings.md) | Semantic search and similarity using on-device sentence embeddings | iOS 13.0+ / macOS 10.15+ |

## Prerequisites

- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)
- [.NET MAUI workload](~/get-started/installation.md)
- A supported Apple device or simulator (see [Requirements](requirements.md))
